# Collection Model

## Purpose

This document is the lightweight content architecture for the static museum.

## Object Fields

Each exhibition object should be trackable with at least:

- title
- date / period
- room
- object type
- curatorial role
- significance note
- image file
- related people / institutions

## Current Object Types

- issue artifact
- production artifact
- cinematic prop display
- portrait card

## Current Rooms

- Opening gallery
- Golden Age Origins
- Marvel Age Expansion
- Crossover and Cinematic Age
- Publication lineage
- People and studios
- Collection highlights

## Curatorial Roles

Objects usually serve one or more of these roles:

- threshold object
- reinvention object
- proof object
- cast object
- scale object
- atmosphere / context object

## Current Collection Notes

The present site already includes:

- origin issue artifacts
- breakthrough issue artifacts
- editorial culture objects
- cinematic hardware objects
- portrait cards for key creators and institutions

Future additions should be added only if they strengthen the chronology spine or a companion room's evidence model.
